# SDLC Model Recommendation System

A smart, web-based tool designed to help Project Managers and Software Engineers select the most appropriate Software Development Life Cycle (SDLC) model for their specific project needs.

## 🧠 How it Works
The system implements a **Decision Tree Logic Engine**. By analyzing four key project dimensions, the algorithm traverses different logic branches to find the optimal fit:

1.  **Requirements Stability:** (Fixed vs. Evolving)
2.  **Project Scale:** (Small vs. Large/Complex)
3.  **Risk Level:** (Standard vs. High-Criticality)
4.  **Client Involvement:** (Minimal vs. Continuous)

## ✨ Key Features
- **Instant Analysis:** Get a recommendation immediately after inputting project parameters.
- **Explainable Results:** Unlike "black-box" systems, this tool provides a step-by-step reasoning path explaining *why* a specific model was chosen.
- **Visual Path Summary:** A breadcrumb-style visualization of the decision-making process.
- **Modern UI:** Built with a clean, responsive interface using Tailwind CSS.

## 🛠️ Tech Stack
- **Frontend:** HTML5, Tailwind CSS
- **Logic Engine:** JavaScript (Conditional Decision Tree implementation)
- **Icons:** FontAwesome

## 📊 Models Supported
- **Waterfall:** For stable, low-risk, small projects.
- **Agile (Scrum):** For evolving requirements with high client involvement.
- **V-Model:** For high-risk projects requiring rigorous verification.
- **Spiral:** For large-scale projects with high technical uncertainty.
- **Incremental:** For large projects with stable requirements needing phased delivery.
- **Kanban:** For continuous maintenance and flow-based delivery.
